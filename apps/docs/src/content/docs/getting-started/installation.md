---
title: Installation
description: Install Minions Invoices packages
---

## TypeScript / Node.js

```bash
npm install @minions-invoices/sdk minions-sdk
```

## Python

```bash
pip install minions-invoices
```

## CLI

```bash
npm install -g @minions-invoices/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
