# Minions Invoices

Invoice generation, payment tracking, due dates, and billing status for agency clients

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-invoices/sdk`) — Core library for Node.js
- **Python SDK** (`minions-invoices`) — Python bindings
- **CLI Tool** (`@minions-invoices/cli`) — Command-line interface
- **Web App** — Interactive playground at [invoices.minions.wtf](https://invoices.minions.wtf)
